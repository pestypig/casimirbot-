# Scaffold Hello Verifier

Local scaffolder for the hello-verifier template.

Usage
```
node tools/scaffold-hello-verifier/index.mjs --dest <path> [--force] [--dry-run]
```

Options
- `--dest <path>`: destination directory (required).
- `--force`: overwrite existing files.
- `--dry-run`: show files without writing anything.

Example
```
node tools/scaffold-hello-verifier/index.mjs --dest ../hello-verifier
```
