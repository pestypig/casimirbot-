# Luma emoticon assets

Keep the seven emotion PNGs here so the server build can serve them at `/luma/emotes/<name>.png`:

- `emoji mad.png`
- `emoji upset.png`
- `emoji shock.png`
- `emoji question.png`
- `emoji happy.png`
- `emoji friend.png`
- `emoji love.png`

If any file is missing the front-end hides the overlay for that mood.
