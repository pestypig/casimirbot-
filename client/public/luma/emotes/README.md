# Luma emoticon assets

Drop the seven PNGs for Luma's moods in this folder. Filenames are fixed so the UI can load them without configuration:

- `emoji mad.png`
- `emoji upset.png`
- `emoji shock.png`
- `emoji question.png`
- `emoji happy.png`
- `emoji friend.png`
- `emoji love.png`

During development the files are served from `/luma/emotes/<name>.png`. If a file is missing, the overlay hides itself.
