# Synthetic Research Notes

Use this folder for generated research-style materials that are useful as
synthetic data, exploratory scaffolding, prompt fuel, or comparison material but
are not canonical papers.

Examples:

- deep-research prompts and generated literature passes
- exploratory sweeps without maintained sidecars
- research-style drafts that are not actively maintained
- speculative cross-domain notes used for ideation or training context

If a document becomes a maintained paper with citations, claim boundaries, and
sidecar support, promote it to `../research/` and keep its sidecars beside it.
