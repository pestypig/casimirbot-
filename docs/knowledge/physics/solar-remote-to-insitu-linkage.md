# Solar Remote-to-In-Situ Linkage

Remote and in-situ solar-wave links are represented as evidence-bearing lineage hypotheses.

## Contract rule

- linkage status: `candidate`, `supported`, or `rejected`
- no hard identity claims for cross-instrument links
- if in-situ observables are attached, lineage must include:
  - frequency-band evidence
  - heliocentric geometry reference

This is a guardrail against overclaiming causality when event timing, layer, and observer geometry differ.
