# Pulsar Dynamic-Spectrum Diffraction

Crab zebra-band features are represented in this lane as dynamic-spectrum transfer observables (`spectrogram`/`channel_series`), not as atomic line-origin claims.

## Contract Semantics

- `feature_kind = zebra_band`
- modality restriction: `spectrogram` or `channel_series`
- response-model path should use transfer-function semantics for magnetospheric propagation/diffraction structure

## Research Basis

- Zebra-band origin interpreted as diffraction/interference structure through plasma-filled magnetosphere transfer paths: [PRL 133, 205201](https://link.aps.org/doi/10.1103/PhysRevLett.133.205201)
