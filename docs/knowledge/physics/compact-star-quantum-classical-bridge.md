# Compact-Star Quantum-Classical Bridge

This bridge records how microphysical hypotheses (pair cascades, gap sparking, EOS class) map to classical observables (timing, polarization, pulse profile, dynamic spectrum).

## Bridge Semantics

- `quantum_side_refs`: closure references and microphysical assumptions
- `classical_side_refs`: observable products and derived constraints
- `bridge_status`: hypothesis / diagnostic / reduced_order / validated
- bridge status does not bypass claim-tier rules on unresolved matter models
