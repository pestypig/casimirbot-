Definition: The potato-radius transition is the size scale at which self-gravity becomes strong enough that material strength can no longer preserve a strongly irregular shape.
Key questions: Is the source talking about the minimum size for rounding or about a specific object class?
Notes: This is a threshold statement, not a precise universal constant. In the current source set, Mimas serves as a near-threshold icy-moon exemplar while Hyperion serves as a clearly irregular low-density counterexample. The repo now also supports both bodies as synthetic Saturn-moon diagnostic profiles so the threshold can be exercised as executable diagnostics without claiming a kernel ephemeris for those moons.
