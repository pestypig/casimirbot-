# Microtubule Energy Migration

Status: exploratory
Node id: `physics-quantum-semiclassical-microtubule-energy-migration-definition`
Lane: quantum-semiclassical

This node is a transport benchmark only and must stay separate from stellar flare and sunquake lanes.

## Exploratory claim
Energy migration in microtubules is an exploratory transport observable: if coherent or semi-coherent migration exists, it must be measured as transport length, anisotropy, damping, and temperature dependence.

## Falsifiers
- no migration length is distinguishable from the noise floor
- anisotropic transport is not reproducible
- the effect cannot be separated from generic thermal diffusion
