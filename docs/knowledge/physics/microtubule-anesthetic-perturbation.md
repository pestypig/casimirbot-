# Microtubule Anesthetic Perturbation

Status: exploratory
Node id: `physics-quantum-semiclassical-microtubule-anesthetic-perturbation-definition`
Lane: quantum-semiclassical

This node is a controlled probe of the observables, not evidence of consciousness by itself.

## Exploratory claim
Anesthetic perturbation is admitted as an exploratory probe when binding changes can be linked to shifts in microtubule transport, vibration, or coherence proxies.

## Falsifiers
- anesthetic exposure does not change the targeted observables
- observable shifts do not track binding or dose
- the effect is overstated as proof of Orch-OR
