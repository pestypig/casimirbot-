Definition: Constitutive response is the reduced bulk relation between stress, strain, pressure, strength, viscosity, elasticity, and friction that follows from the effective microphysics layer.
Key questions: Which state variables are kept explicit in the closure, and which regime variables are absorbed into the material law?
Notes: This is the shared closure node for granular bodies, hydrostatic figures, and any reduced material model used downstream.
