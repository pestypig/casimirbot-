# Long-Period Radio Transient Limit

Long-period radio transient and slow-pulsar cases are represented as explicit limit probes and bridge cases rather than immediate model rejection or immediate model replacement.

## Contract Semantics

- use `observed_status = bridge_case` for empirically tense but unresolved cases
- keep baseline closure models declared (death-line, gap, pair-cascade)
- store additional mechanism proposals as separate hypotheses with evidence refs
