# Solar Flare Phase Definition (Observable Contract)

`flare_phase` is an observational context label used for comparison slicing:

- `precursor`
- `impulsive`
- `decay`

It does not claim a unique physical heating mechanism by itself. Mechanism is represented separately in `origin_hypotheses` or forward-model metadata.

The initial DKIST/ViSP flare benchmark in this lane is a decay-phase window for a C6.7 event, which motivates keeping phase explicit in the contract.
