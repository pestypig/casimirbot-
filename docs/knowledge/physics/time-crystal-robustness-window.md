# Time Crystal Robustness Window

Status: exploratory
Node id: `physics-quantum-semiclassical-time-crystal-robustness-window-definition`
Lane: quantum-semiclassical

This node keeps the perturbation window explicit and does not promote a transient resonance to time-crystal order.

## Exploratory claim
A time crystal is only admissible when the subharmonic response survives a finite robustness window against disorder, dissipation, and drive changes.

## Falsifiers
- small perturbations destroy the subharmonic response
- the robustness window collapses under repeated measurement
- the effect is not distinguishable from a transient resonance
