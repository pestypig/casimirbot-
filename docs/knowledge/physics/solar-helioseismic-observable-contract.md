# Solar Helioseismic Observable Contract

Helioseismic observables are first-class event siblings of line and magnetic observables.

Supported modalities:

- `time_distance`
- `egression_map`

Required linkages:

- source reference
- geometry coordinates
- method metadata (direct Doppler/time-distance vs acoustic holography path)

## Source basis

- Sunquake wavefront and time-distance diagnostics overview: https://link.springer.com/article/10.1007/s11207-025-02480-6
- Early-impulsive sunquake source and timing constraints: https://arxiv.org/abs/1804.06565
