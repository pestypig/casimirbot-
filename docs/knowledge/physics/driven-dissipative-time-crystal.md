# Driven Dissipative Time Crystal

Status: exploratory
Node id: `physics-quantum-semiclassical-driven-dissipative-time-crystal-definition`
Lane: quantum-semiclassical

This node stays exploratory and must not be promoted into a maturity claim for microtubules or spacetime geometry.

## Exploratory claim
A driven-dissipative time crystal is admissible only when the system shows stable subharmonic order, openness or dissipation, and a robustness window against perturbations.

## Math hook
The minimal diagnostic signature is a stable subharmonic response relative to the drive period.

## Falsifiers
- there is no subharmonic response
- the effect disappears once drive and dissipation are modeled correctly
- the node is used to certify a biological clock without a time-crystal signature
