# Solar Photospheric Line Transient Definition

Photospheric line transients are modeled as radiative observables linked to event bundles:

- baseline reference
- line-center shift
- width and asymmetry tracks
- equivalent-width evolution
- optional Stokes mode

Line transients do not carry source anisotropy or ribbon-motion geometry fields. Those belong in geometry/impact objects.
