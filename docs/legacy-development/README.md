# Legacy Development Notes

Use this folder for preserved development history: approaches that were tried,
superseded, contradicted by later evidence, or no longer match the active code.

These files can still be valuable. They explain why a path was attempted, what
failed, and what constraints shaped the current system. They should not be
presented as current implementation truth.

Before moving a file here, check whether a newer document supersedes it and add
a short note or index entry that points readers to the current source.
