Good catch — the “no visible change” is very likely because gammaGeo never reaches the engine for the REAL (parity) pane. In HelixCore you put it on g_y, but in WarpBubbleCompare the shader scale (thetaScale) is computed from gammaGeo, not g_y. Since that prop mismatch makes gammaGeo fall back to the default 26, your mode changes barely move the needle (and with parity exposure/zeroStop, they can look identical).

Here’s a tight set of fixes + a quick verification routine.

1) Pass the right props from HelixCore

In HelixCore, when building compareParams, add gammaGeo (and it’s fine to keep g_y for backward compat). Also pass an explicit sectors so the scale uses the right divisor.

// before
const compareParams = useMemo(() => {
  const base: any = {
    currentMode: effectiveMode,
    hull,
    lightCrossing: lc,
    sectorStrobing: sectorsResolved,

    dutyCycle:        num(pipeline?.dutyCycle)        ?? 0.14,
    g_y:              num(pipeline?.gammaGeo)         ?? 26,      // <- only g_y
    qSpoilingFactor:  num(pipeline?.qSpoilingFactor)  ?? 1,
    gammaVanDenBroeck: num(pipeline?.gammaVanDenBroeck) ?? 2.86e5,
    dutyEffectiveFR,
    powerAvg_MW:      num(pipeline?.P_avg)            ?? 83.3,
    exoticMass_kg:    num(pipeline?.M_exotic)         ?? 1405,
  };
  return base;
}, [/* … */]);

// after
const compareParams = useMemo(() => {
  const base: any = {
    currentMode: effectiveMode,
    hull,
    lightCrossing: lc,
    sectorStrobing: sectorsResolved,
    sectors: sectorsResolved,                    // <- explicit

    dutyCycle:        num(pipeline?.dutyCycle)        ?? 0.14,
    gammaGeo:         num(pipeline?.gammaGeo)         ?? 26,      // <- add this
    g_y:              num(pipeline?.gammaGeo)         ?? 26,      // (keep for compat)
    qSpoilingFactor:  num(pipeline?.qSpoilingFactor)  ?? 1,
    gammaVanDenBroeck: num(pipeline?.gammaVanDenBroeck) ?? 2.86e5,
    dutyEffectiveFR,
    powerAvg_MW:      num(pipeline?.P_avg)            ?? 83.3,
    exoticMass_kg:    num(pipeline?.M_exotic)         ?? 1405,
  };
  return base;
}, [/* … */]);
