Sector semantics mix-up (can double-average duty)
toSharedUniforms sets sectors from sectorCount when sectorStrobing/sectors are missing. In your engine, sectors = concurrent strobing; sectorCount = total. Feeding sectorCount into sectors will over-average.

Fix:

-  return {
+  const sectorsLive  = Math.max(1, Math.floor(N(snap.sectorStrobing ?? snap.sectors, 1)));
+  const sectorsTotal = Math.max(1, Math.floor(N(snap.sectorCount ?? sectorsLive, sectorsLive)));
+  return {
     ...
-    sectors: Math.max(1, Math.floor(N(snap.sectorCount ?? snap.sectorStrobing ?? snap.sectors, 1))),
+    sectors: sectorsLive,
+    sectorCount: sectorsTotal,
     split: Math.max(0, Math.floor(N(snap.sectorSplit ?? snap.split ?? snap.currentSector, 0))),
