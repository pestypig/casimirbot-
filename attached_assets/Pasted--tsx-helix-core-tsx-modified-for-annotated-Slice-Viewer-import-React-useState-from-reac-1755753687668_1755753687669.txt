- **Removed Slice Preferences panel** — no sliders/toggles, purely visual for now.
- **Added overlay annotations**:

  - Y-axis = curvature κ(ρ) (per m²).
  - Grid scale annotation (example: 50 m per major division; you can adjust if your grid spacing is different).
- **Non-white text**: annotations styled in `text-slate-700` with semi-transparent background for contrast against the white slice.
