2) Seed hull framing + θ-scale during bootstrap (first frame)

Inside your makeEngine function, right before you build uniforms, derive axes + span and fix the VdB default:

// Derive seeded axes/clip/span from hull (prevents first-frame NaNs/black)
const a = num(hull.a, 503.5), b = num(hull.b, 132), c = num(hull.c, 86.5);
const s = 1 / Math.max(a, b, c, 1e-9);
const axesSceneSeed: [number,number,number] = [a*s, b*s, c*s];
const spanSeed = Math.max(VIS.minSpan, Math.max(...axesSceneSeed) * VIS.spanPaddingDesktop);

// Harden gammaVdB default (match AmplificationPanel & pipeline)
const gammaVdBDefault = num(parameters.gammaVanDenBroeck, 2.86e5);


Then build the bootstrap uniforms with these seeded values + parity-aware boosts:

const uniforms = {
  // …(existing)…
  curvatureGainDec: parity ? 0 : 3,
  curvatureBoostMax: parity ? 1 : (parameters.curvatureBoostMax ?? 40),
  curvatureGainT: parity ? 0 : (parameters.viz?.curvatureGainT ?? parameters.curvatureGainT ?? 0),

  dutyCycle: dutyResolved,
  gammaGeo,
  Qburst: qCavity,
  deltaAOverA: qSpoil,
  gammaVdB: gammaVdBDefault,           // << was 3.83e1; now 2.86e5 default

  // Seed framing so engine has non-null axes in first frame
  axesScene: axesSceneSeed,
  axesClip:  axesSceneSeed,
  hullAxes:  [a,b,c] as [number,number,number],
  gridSpan:  parameters.gridSpan ?? spanSeed,

  wallWidth: wallNorm,
  physicsParityMode: !!parameters.physicsParityMode,

  epsilonTilt: epsilonTiltResolved,
  betaTiltVec: betaTiltResolved,
  tiltGain: tiltGainResolved,
};


Right after engine.bootstrap(uniforms), push the unified θ-scale immediately (so first frame renders with correct amplitude), and keep your existing viz overrides:

// Prime θ-scale for the first frame (single source of truth)
engine.updateUniforms({ thetaScale: resolveThetaScale({
  dutyCycle: dutyResolved,
  sectors: sectorsResolved,
  gammaGeo,
  qSpoilingFactor: qSpoil,
  gammaVdB: gammaVdBDefault
})});


You already call engine.updateUniforms({...viz}) after bootstrap—keep that as-is.
