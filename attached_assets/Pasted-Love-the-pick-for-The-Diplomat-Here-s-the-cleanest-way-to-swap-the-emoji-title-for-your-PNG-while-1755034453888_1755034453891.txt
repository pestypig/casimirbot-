2) If your start page is React ‚Äì make the card use an <img>

Update your ProfileCard (or wherever you render each profile) to accept an imgSrc and render it. Keep the label for screen readers, but visually hide it so the PNG is the face of the card.

// client/src/components/ProfileCard.tsx
type ProfileCardProps = {
  id: "optimist" | "engineer" | "diplomat" | "strategist";
  title: string;
  imgSrc: string;        // <-- new
  onSelect: (id: ProfileCardProps["id"]) => void;
};

export function ProfileCard({ id, title, imgSrc, onSelect }: ProfileCardProps) {
  return (
    <button
      onClick={() => onSelect(id)}
      className="profile-card"
      aria-label={`Choose ${title}`}
    >
      <img
        src={imgSrc}
        alt=""                 // image is decorative; text label is below for a11y
        loading="lazy"
        className="profile-img"
      />
      <span className="sr-only">{title}</span>
    </button>
  );
}

Use it like:

<ProfileCard
  id="diplomat"
  title="The Diplomat"
  imgSrc="/profiles/diplomat.png"   // <-- your PNG
  onSelect={handleSelect}

3) Minimal CSS so the PNG looks great and the whole card is clickable

/* Somewhere globally or in the Start page CSS */
.profile-card {
  display: grid;
  place-items: center;
  width: 220px;               /* match your layout */
  height: 180px;
  border-radius: 18px;
  background: rgba(255,255,255,0.03);
  border: 1px solid rgba(255,255,255,0.08);
  transition: transform .15s ease, box-shadow .15s ease, border-color .15s ease;
  cursor: pointer;
}

.profile-card:hover,
.profile-card:focus-visible {
  transform: translateY(-2px);
  border-color: rgba(130, 200, 255, 0.45);
  box-shadow: 0 8px 28px rgba(0,0,0,0.35), inset 0 0 0 1px rgba(130,200,255,0.25);
  outline: none;
}

.profile-img {
  width: 92px;                /* tweak per image */
  height: 92px;
  object-fit: contain;
  filter: drop-shadow(0 8px 16px rgba(0,0,0,.35));
  user-select: none;
  pointer-events: none;       /* ensures the BUTTON gets the click, not the img */
}

/* visually hidden (keeps text for screen readers) */
.sr-only {
  position: absolute !important;
  width: 1px; height: 1px;
  padding: 0; margin: -1px; overflow: hidden;
  clip: rect(0 0 0 0); white-space: nowrap; border: 0;
}

That‚Äôs it‚Äîthe PNG becomes the face of the profile, the entire tile is clickable, and accessibility is preserved.
