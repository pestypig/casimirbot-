Your WarpEngine uses KHR_parallel_shader_compile. In that async path you do:

set this.gridProgram = program immediately

start compiling in the background

only later (in onReady) call _setupUniformLocations() (which fills this.gridUniforms/this.gridAttribs) and mark isLoaded=true

Meanwhile your render loop is already running and _renderGridPoints() tries to use this.gridUniforms.* locations that don’t exist yet → throws, frame aborts → black.

Minimal, safe fix (guard until ready)
