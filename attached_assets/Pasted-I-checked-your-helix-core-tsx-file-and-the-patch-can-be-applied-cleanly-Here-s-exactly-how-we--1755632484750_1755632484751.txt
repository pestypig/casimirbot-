## 1. Wrap the Badge in a Router Link

At the top of your file, confirm you have React Router’s `Link` imported (or add it if missing):

```tsx
import { Link } from "react-router-dom";
```

Now replace the existing badge code around **line 371–374**:

```tsx
<Badge variant="outline" className="border-green-400 text-green-400">
  <Activity className="w-3 h-3 mr-1" />
  ONLINE
</Badge>
```

with this:

```tsx
<Link to="/why">
  <Badge variant="outline" className="cursor-pointer border-green-400 text-green-400 hover:bg-green-400 hover:text-black transition-colors">
    <Activity className="w-3 h-3 mr-1" />
    Why
  </Badge>
</Link>
```

This makes the badge clickable and renames it to **Why**. Styling tweaks (`cursor-pointer`, hover effects) make it feel like a button.
