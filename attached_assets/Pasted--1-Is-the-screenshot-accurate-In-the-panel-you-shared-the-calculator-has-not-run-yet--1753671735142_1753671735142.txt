### 3  |  Implementation steps

1. **Run a simulation** – ensure the Results card is populated.
2. Read those values in `app.py`:

   ```python
   E_flat   = results['static_energy']      # −2.55e-3 J
   E_bowl   = results['bowl_energy']
   gamma    = (E_bowl/E_flat)**(1/3)
   E_Q      = E_bowl * Q
   E_duty   = E_Q   * duty
   E_deff   = E_duty* duty_eff
   power    = peak_power * duty_eff
   zeta     = zeta_value
   ```
3. Feed them into the three Plotly figures above.
4. Add a **“Download PDF Proof”** button (`st.download_button`) that saves the ledger plus the plots to a single `report.pdf` (reportlab or weasyprint).
